# Chrome Extension TypeScript Starter

Chrome Extension, React, TypeScript 

## Prerequisites

* [node + npm](https://nodejs.org/) (Current Version)


## Includes the following

* TypeScript
* Webpack
* React
* Tailwind
* Example Code
    * Chrome Storage
    * Options Version 2
    * content script
    * count up badge number
    * background

## Setup

```
npm install
```
